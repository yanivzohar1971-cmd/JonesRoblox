## What was configured

default.project.json syncs only these services:

- ReplicatedStorage → src/ReplicatedStorage
- ServerScriptService → src/ServerScriptService
- StarterPlayer.StarterPlayerScripts → src/StarterPlayer/StarterPlayerScripts
- StarterGui → src/StarterGui

**Workspace is not included** — Studio owns the world.

[Jones In The Lane] Server initialized via Rojo.


## Commands to run

**1. Start Rojo** (from the project folder):


powershell
cd c:\Users\Yaniv\source\repos\JonesRoblox
rojo serve


**2. In Roblox Studio** (with your place open):

1. OpenBorder the [Rojo plugin](https://rojo.space/docs/v7/getting-started/installation/#install-the-plugin) if you haven't already.
2. Open the Rojo plugin panel.
3. Click **Connect** → localhost:34872 (default).
4. Click **Sync**.

**3. Test:** Press **Play** in Studio and check **Output** for the startup message.

---

**Reminder:** Studio owns visuals and the map; Rojo owns code only. Edit scripts under src/, save, and sync again while rojo serve is running.

## Do this first in Studio (right now)

You’re in **Edit** mode (not Play). Rojo is already synced.

### 1. Insert the Industrial zone

1. Top ribbon → **Home** tab  
2. Click **Part** → **Block**  
3. **Explorer** (right) → **Workspace** → select the new Part  
4. Press **F2** → rename to **Zone_Industrial**

### 2. Move and resize it

1. Toolbar → **Move** (four arrows) or **Ctrl+2** → drag it to where “work” should happen on the ground  
2. Toolbar → **Scale** (cube with balls) or **Ctrl+3** → make it **large** (roughly 20×10×20 studs or bigger)

### 3. Set properties

With Zone_Industrial selected → **Properties** panel (View → Properties if missing):

- **Anchored** → ✅ **on**  
- **CanCollide** → ☐ **off**  
- **Transparency** → **0.5** (so you can see the box)

### 4. Add the tag

1. **View** → **Tag Editor**  
2. Select Zone_Industrial in Explorer  
3. Tag Editor → **Add tag**  
4. Type exactly: **JonesZone_Industrial** → Enter  

### 5. Test immediately

1. Press **Play** (green ▶ or **F5**)  
2. Walk **into** the semi-visible box  
3. HUD should show **Current Zone: Industrial** and a **Work Shift** button  

If that works, repeat steps 1–4 for **Zone_Home** with tag **JonesZone_Home**, then test **Rest**.
