# Game Concept

## Name
Jones In The Lane

## Core Idea
A web-based 2D life simulation game where the player lives inside a city lane and builds a life through work, housing, education, social relationships, banking, risk, reputation, business, inventory, health, and events.

## Style
- 2D simulation
- React UI + Canvas map
- Web first
- Mobile friendly later
- Not 3D
- Not overcomplicated rendering

## Inspiration Direction
- RimWorld-style systems depth
- GTA2 / top-down map feeling
- The Sims-style life needs and progress
- Tycoon / management panels

## Player Loop
1. Move around the map
2. Enter zones
3. Choose actions
4. Earn money / lose energy / gain reputation
5. Unlock education, jobs, housing, business
6. React to life events and economy changes
7. Grow from simple citizen to advanced city actor

## Main Zones
- Home
- Downtown
- Market
- School
- Industrial

## First Actions
- Rest
- Browse items
- Buy food
- Enroll course
- Search jobs
- Visit bank
- Work shift
